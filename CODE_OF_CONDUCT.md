# Code of Conduct

This is a practice space for people learning open source collaboration
for the first time. A few simple expectations:

- Be patient and kind, especially with people opening their first-ever
  pull request.
- Give feedback on code, not on people.
- Assume good intent — most mistakes here are learning moments, not
  carelessness.
- No harassment, discrimination, or personal attacks, ever.

If something feels off, raise it with the maintainer directly.
